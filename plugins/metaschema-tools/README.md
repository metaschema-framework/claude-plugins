# Metaschema Tools Plugin

Commands and MCP servers for [metaschema-java](https://github.com/metaschema-framework/metaschema-java) CLI integration.

## Installation

```bash
claude plugin install metaschema-framework:metaschema-tools
```

## Requirements

- [metaschema-java](https://github.com/metaschema-framework/metaschema-java) installed and available in PATH

## Skills

### using-metaschema-java

Guide to metaschema-java CLI commands:

- Module validation
- Content validation against Metaschema
- XML/JSON schema generation
- Format conversion

### metaschema-java-library

Comprehensive guide to using the metaschema-java library programmatically:

- Module structure and key packages
- Loading Metaschema modules
- Metapath expression evaluation
- Serialization/deserialization (XML, JSON, YAML)
- Constraint validation
- Exception hierarchy and error handling
- Implementing Metapath functions
- Binding annotations and code generation
